# 🎵 音乐文件说明

## 📂 音乐文件放置位置
请将音乐文件放置在以下目录中：
```
H:\Nitrox\Assets\Music\
```

## 🎼 支持的音乐格式
- **MP3** (*.mp3) - 推荐格式，兼容性最好
- **WAV** (*.wav) - 高质量无损格式
- **OGG** (*.ogg) - 开源格式，质量好

## 📝 文件命名建议
- 中文文件名：`贡献者主题音乐.mp3`
- 英文文件名：`contributors_theme.mp3`
- 避免特殊字符：不要使用 `/ \ : * ? " < > |` 等字符

## 🔧 使用说明
1. 将音乐文件拷贝到 `Assets/Music/` 文件夹
2. 启动Nitrox启动器
3. 进入"工具"页面
4. 在"社区贡献者"区域点击"🎵 播放音乐"按钮
5. 系统会自动播放第一个找到的音乐文件

## 📋 示例文件结构
```
H:\Nitrox\
├── Assets/
│   └── Music/
│       ├── 贡献者主题音乐.mp3
│       ├── contributors_bgm.wav
│       └── nitrox_theme.ogg
├── Nitrox.Launcher/
└── ...
```

## ⚠️ 注意事项
- 确保音乐文件不过大（建议小于10MB）
- 音乐将从第一个找到的文件开始播放
- 如果没有找到音乐文件，系统会提示用户放置音乐文件
- 音乐播放功能需要系统支持对应的音频格式

## 🎯 推荐设置
- **文件大小**：2-5MB
- **比特率**：128-320 kbps
- **时长**：30秒-3分钟
- **音量**：适中，避免过大或过小

---
**注意**：音乐文件由用户自行提供，请确保拥有使用权限。
